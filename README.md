### PHP Script Descriptions

- **bankBalance.php / bankBalanceBetter.php**
  - Implements a recurring menu for managing bank balances.

- **notas.php**
  - Calculates the average of input values and iterates through a loop to check if the average is greater than 6.

- **numbers.php**
  - Takes all the arguments passed during execution, creates an array, and removes duplicate elements.

- **test.php**
  - A file used for testing random code snippets.
